# config

Display Bld's full configuration, including options with default values that you haven't explicitly set.

## Options

| Flag | Short | Description |
|------|-------|-------------|
| `--edit` | `-e` | Open the config file in your default editor |
| `--help` | `-h` | Print help |
| `--version` | `-V` | Print version |

## Examples

Print the current configuration:

```bash
$ bld config
```

Open the config file for editing:

```bash
$ bld config -e
```
